How to debug this?

To enable full debug logging, set the NODE_ENV environment variable to debug.

How to look at the server logs?

If you're using Docker, you can follow these steps:

  1. List all containers with docker ps -a to get pigallery2's container.
  2. Get logs for the pigallery2 container: docker logs -f pigallery2

For non-Docker setups, check the standard output of the Node.js process.

How to look at the client logs?

In most browser right->click inspect should bring up the developer console.
